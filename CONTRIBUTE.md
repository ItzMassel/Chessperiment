# How to contribute

Start by visiting the [canban board](https://github.com/users/ItzMassel/projects/1/views/1). Pick one issue to work on (TIP: Look out for the tag "good first issue"). 

Then visit this page: [https://github.com/ItzMassel/Chessperiment](https://github.com/ItzMassel/Chessperiment). Click the `fork`-button. Leave the settings as they are and hit "Fork Repository". You've now copied the original repository. You can start coding! If you've never worked with Github before, check out [this tutorial](https://www.youtube.com/watch?v=v5gnvDUWqFM&t=28s). Once your done, test your fix and open a pr to the alpha branch. The team (which is just me for now) will review it, and then merge it. Once there enough updates in the alpha branch, the branch will be tested and merged into beta, which will then be merged into main, meaning the changes will be live.

If you have any questions, use the help channel on the discord or dm me or the team (which doenst exist... yet.).
